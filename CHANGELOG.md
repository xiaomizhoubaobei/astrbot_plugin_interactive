# 变更日志 (Changelog)

所有值得注意的变更都将记录在此文件中。

格式遵循[Keep a Changelog](https://keepachangelog.com/en/1.0.0/)标准，并且该项目采用[语义化版本控制](https://semver.org/lang/zh-CN/)。

## [v0.0.3] - 2026-02-28

### ✨ 新功能
- 添加游戏配置模块 - 新增了完整的配置管理系统，支持各项游戏参数的自定义配置
- 添加文档生成提示以适配 AstrBot 插件风格 - 改进了插件的文档兼容性
- 实现积分系统核心功能 - 包括积分获取、消费和管理机制

### 📝 文档
- 添加游戏系统功能文档 - 详细介绍了各项游戏功能的操作方式和规则
- 更新插件使用说明

### 🔧 杂项
- 更新 GitHub Actions 工作流配置和中文翻译 - 改进了CI/CD流程

## [v0.0.2] - 2026-02-21

### ✨ 新功能
- 实现猜数字游戏 - 用户可以参与数字猜测游戏获得积分奖励
- 实现每日签到系统 - 用户可以每日签到获得基础积分奖励
- 实现积分抽奖系统 - 消耗积分进行抽奖，有概率获得高价值奖励
- 实现积分商店系统 - 用户可以用积分购买各种道具和功能卡
- 实现物品使用系统 - 支持使用各种功能性道具
- 实现用户个人资料查看 - 展示用户的游戏进度和统计数据
- 实现成就系统 - 完成特定目标解锁成就
- 实现牛牛养成系统 - 用户可以领养和照顾虚拟宠物牛牛
- 实现幸运转盘系统 - 额外的积分消耗和奖励机制

### 🐛 修复
- 修复各种游戏逻辑漏洞
- 优化错误处理机制

### 📝 文档
- 完善用户使用指南
- 添加API文档
