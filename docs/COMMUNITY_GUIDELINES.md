# AstrBot 互动游戏插件 - 社区参与指南

## 反馈机制

### 用户反馈渠道
- [ ] GitHub Issues
  - Bug报告
  - 功能请求
  - 问题讨论
- [ ] 社区论坛
  - 使用经验分享
  - 建议收集
  - 问题解答
- [ ] 用户调查
  - 定期满意度调查
  - 功能优先级调查
  - 新功能需求调研

### 反馈处理流程
1. 收集用户反馈
2. 分类和标记
3. 评估优先级
4. 纳入开发计划
5. 反馈处理结果

### 反馈响应时间
- Bug报告：24小时内响应
- 功能请求：3天内评估
- 一般问题：48小时内回复
