# 礼物赠送系统

## 功能概述
礼物赠送系统允许用户之间互相赠送积分、物品或虚拟礼品，增加玩家间的互动和社交性。

## 命令
- `/gift [user] [amount] [item]` - 向指定用户赠送积分或物品
- `/gift_list` - 查看收到的礼物
- `/accept_gift [gift_id]` - 接收礼物
- `/decline_gift [gift_id]` - 拒绝礼物

## 功能特性
- 支持多种赠送类型（积分、物品、礼包）
- 发送后需接收方确认
- 可设置赠送留言
- 防止恶意赠送（赠送冷却时间）

## 限制与规则
- 单次赠送最高限额
- 每日赠送次数限制
- 赠送双方需要有一定关系（好友、公会等）
- 部分物品不可赠送

## 通知系统
- 发送时通知接收方
- 自动提醒待接收的礼物
- 赠送成功/失败反馈

## 手续费
- 大额赠送收取少量手续费
- VIP用户享有免费赠送额度
