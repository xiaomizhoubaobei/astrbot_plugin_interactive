# 成就徽章墙

## 功能概述
成就徽章墙展示用户获得的各种成就和徽章，激励用户探索游戏的不同方面，提供收藏和展示功能。

## 命令
- `/badges` - 查看所有徽章
- `/badge_display [badge_id]` - 设置展示徽章
- `/badge_album` - 查看徽章相册

## 功能特性
- 徽章分类展示
- 稀有度分级
- 收集进度追踪
- 个性化展示

## 徽章类型
- 里程碑类：达到特定数值解锁（如"百发百中"：抽奖100次）
- 技能类：完成特定挑战解锁（如"签到达人"：连续签到30天）
- 探索类：发现隐藏功能解锁（如"寻宝专家"：找到所有隐藏宝藏）
- 社交类：与他人互动解锁（如"社交之星"：赠送物品100次）
- 隐藏类：特殊条件解锁（如"幸运儿"：连续3次抽中SR）

## 徽章效益
- 解锁特殊称号
- 获得额外奖励
- 展示个人成就
- 影响排行榜权重

## 个性化设置
- 自定义徽章展示顺序
- 设置徽章展示隐私
- 徽章分享功能
- 动态徽章效果
