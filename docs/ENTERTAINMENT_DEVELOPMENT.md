#### 2.4 商店推荐系统

**目标**: 智能推荐商品

**实施内容**:

**推荐策略**:
1. **基于消费**: 根据历史购买推荐
2. **基于等级**: 根据用户等级推荐
3. **基于活动**: 推荐活动商品
4. **基于稀缺**: 推荐库存紧张商品
5. **基于喜好**: 根据游戏偏好推荐

**推荐算法**:
```python
def recommend_items(self, user_data: Dict, user_history: List) -> List[Dict]:
    """推荐商品
    
    Args:
        user_data: 用户数据
        user_history: 用户历史记录
    
    Returns:
        推荐商品列表
    """
    recommendations = []
    
    # 基于用户等级推荐
    level = user_data.get("level", 1)
    if level < 3:
        recommendations.append({"item": "新手礼包", "reason": "适合新手"})
    elif level >= 5:
        recommendations.append({"item": "幸运护符", "reason": "提升运气"})
    
    # 基于游戏偏好推荐
    if "guess" in user_data.get("favorite_games", []):
        recommendations.append({"item": "提示令牌", "reason": "猜数字必备"})
    
    # 基于活动推荐
    current_event = self._get_current_event()
    if current_event:
        recommendations.append({
            "item": current_event["special_item"],
            "reason": "限时活动"
        })
    
    return recommendations[:3]
```

- `redpacket <总积分> <数量>` - 发普通红包
- `redpacket lucky <总积分> <数量>` - 发拼手气红包
- `redpacket open <红包ID>` - 抢红包

**红包数据**:
```python
{
    "id": "rp_123",
    "type": "lucky",                 # 普通红包/拼手气红包
    "total_points": 100,
    "count": 5,
    "sender": "user1",
    "recipients": [],                # 已领取用户
    "remaining": 5,                  # 剩余数量
    "status": "active",              # active/expired
    "created_time": 1708569600000,
    "expires_time": 1708656000000    # 24小时后过期
}
```
