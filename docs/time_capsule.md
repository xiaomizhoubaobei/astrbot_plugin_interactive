# 时间胶囊

## 功能概述
时间胶囊系统允许用户写入信息、愿望或内容，设置在未来某个时间点开启，增加情感连接和期待感。

## 命令
- `/capsule_create [date] [message]` - 创建时间胶囊
- `/my_capsules` - 查看我的胶囊
- `/capsule_open [capsule_id]` - 开启时间胶囊

## 功能特性
- 定时开启机制
- 加密存储
- 提醒功能
- 分享机制

## 胶囊类型
- 个人胶囊：仅自己可见
- 公开胶囊：到期后公开
- 友情胶囊：与好友共享
- 社区胶囊：全体用户可查看

## 创建内容
- 文字信息
- 图片附件
- 目标计划
- 情感寄语

## 开启奖励
- 到期自动提醒
- 开启获得积分奖励
- 连续使用奖励
- 特殊纪念日额外奖励
