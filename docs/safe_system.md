# 积分保险箱

## 功能概述
积分保险箱系统允许用户存放部分积分，防止因游戏风险而损失，保障用户资产安全。

## 命令
- `/safe` - 查看保险箱
- `/safe_deposit [amount]` - 存入保险箱
- `/safe_withdraw [amount]` - 取出保险箱
- `/safe_settings` - 设置保险箱

## 功能特性
- 安全存储积分
- 提取时间限制
- 存款利率
- 保险箱等级

## 存款规则
- 最低存款金额
- 存款时间锁定
- 分批提取限制
- VIP额外存储空间

## 保险箱等级
- 初级保险箱：1000积分容量
- 高级保险箱：5000积分容量
- 贵族保险箱：20000积分容量
- 皇家保险箱：100000积分容量

## 增值服务
- 存款获得利息
- 长期存款奖励
- 保险箱安全保障
- 意外损失赔偿
