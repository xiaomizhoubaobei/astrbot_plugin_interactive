# 竞猜系统

## 功能概述
竞猜系统允许用户对各种事件（真实或虚拟）下注积分，猜中获得奖励，增加游戏的紧张感和参与度。

## 命令
- `/bet` - 查看当前竞猜
- `/bet_place [event_id] [choice] [amount]` - 下注
- `/bet_create` - 创建竞猜（权限限制）
- `/my_bets` - 查看我的竞猜

## 功能特性
- 多样化竞猜主题
- 动态赔率
- 实时结算
- 竞猜历史记录

## 竞猜类型
- 体育赛事：足球、篮球等比赛结果
- 虚拟事件：系统设定的虚拟事件
- 社区话题：用户发起的公共话题
- 系统事件：天气、运势等系统事件

## 赔率机制
- 基于投注比例动态调整
- 早期投注享受更好赔率
- 高风险高回报
- 最小赔率保障

## 风险控制
- 单次投注限额
- 累计投注限额
- 冷静期机制
- 防沉迷提醒
