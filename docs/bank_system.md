# 积分银行

## 功能概述
积分银行系统允许用户存款获得利息，提供金融储蓄功能，增强用户粘性和积分流动性。

## 命令
- `/bank` - 查看银行账户
- `/deposit [amount]` - 存款
- `/withdraw [amount]` - 取款
- `/bank_interest` - 查看利息收益

## 功能特性
- 存款获得利息
- 复利计算
- 存款等级
- VIP特权

## 利率机制
- 普通用户：年利率2%
- VIP用户：年利率5%
- 大额存款：年利率8%
- 长期存款：年利率10%

## 存款等级
- 青铜储户：0-10000积分存款
- 白银储户：10000-50000积分存款
- 黄金储户：50000-200000积分存款
- 钻石储户：200000+积分存款

## 特色服务
- 自动续存
- 定期存款
- 存款保险
- 银行贷款
