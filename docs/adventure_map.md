# 冒险地图

## 功能概述
冒险地图系统提供虚拟世界探索功能，用户可以在不同区域冒险，发现宝藏、完成任务、遇见奇遇。

## 命令
- `/adventure` - 开始冒险
- `/adventure_map` - 查看地图
- `/travel [location]` - 前往地点
- `/adventure_log` - 查看冒险日志

## 功能特性
- 多区域地图
- 随机事件
- 探索奖励
- 地图解锁

## 地图区域
- 新手村：安全区域，适合新手
- 森林区：怪物较多，掉落物品
- 沙漠区：特殊事件，稀有物品
- 雪山区：高难度区域，稀有奖励
- 海岛区：海洋探险，独特奖励
- 遗迹区：古代遗迹，珍贵宝物
- 龙谷区：终极挑战，传奇物品

## 冒险机制
- 体力消耗
- 随机遇敌
- 任务完成
- 资源收集

## 奖励系统
- 探索奖励
- 击败敌人奖励
- 寻宝奖励
- 任务完成奖励
