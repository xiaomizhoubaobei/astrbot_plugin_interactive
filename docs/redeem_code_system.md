# 兑换码系统

## 功能概述
兑换码系统允许管理员或活动发放特殊兑换码，用户输入后可获得奖励，常用于活动奖励、推广优惠等。

## 命令
- `/redeem [code]` - 使用兑换码
- `/my_codes` - 查看已使用的兑换码

## 功能特性
- 多种类型兑换码
- 限时有效
- 限制使用次数
- 防复制机制

## 兑换码类型
- 新手礼包码：新人专属，仅限新用户使用
- 节日祝福码：特定节日发放
- 活动奖励码：完成活动后获得
- VIP专享码：仅VIP用户可使用
- 邀请奖励码：邀请新用户后获得

## 技术实现
- 随机生成安全兑换码
- MD5加密存储
- 有效期控制
- 使用次数限制

## 管理后台
- 生成批量兑换码
- 查询兑换码使用情况
- 撤销未使用兑换码
- 导出使用统计报表
