# 聊天室系统

## 功能概述
聊天室系统提供用户间的实时文字交流空间，增强社区互动和游戏体验。

## 命令
- `/chatroom` - 进入公共聊天室
- `/chatroom_create [name]` - 创建私人聊天室
- `/chatroom_join [room_id]` - 加入指定聊天室
- `/chatroom_leave` - 离开聊天室

## 功能特性
- 公共聊天室
- 私人房间
- 房间管理员
- 聊天记录

## 聊天室类型
- 公共大厅：所有人可加入
- 主题房间：特定话题讨论
- 公会频道：公会内部交流
- 好友私聊：两人专属聊天

## 管理功能
- 房间权限设置
- 用户禁言/踢出
- 敏感词过滤
- 举报系统

## 互动功能
- 表情包发送
- 游戏内消息分享
- 聊天室活动
- 话题标签
