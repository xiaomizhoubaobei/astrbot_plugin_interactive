# 个人商店

## 功能概述
个人商店允许用户开设自己的虚拟店铺，出售个人拥有的物品或服务，促进玩家间交易。

## 命令
- `/mystore` - 查看自己的商店
- `/mystore_open` - 开启个人商店
- `/mystore_item [item] [price]` - 上架物品
- `/mystore_remove [item_id]` - 下架物品
- `/shop [user]` - 查看他人商店

## 功能特性
- 个性化店铺装饰
- 自定义定价
- 店铺等级系统
- 营业统计

## 店铺管理
- 商品上下架管理
- 库存监控
- 价格调整
- 促销活动设置

## 收益与费用
- 销售收入100%归店主
- 上架手续费：每次上架收费1积分
- 店铺维护费：每日收取
- 高级店铺特权（付费）

## 信誉系统
- 顾客评价
- 好评率计算
- 优质商家认证
- 信誉等级提升店铺功能
