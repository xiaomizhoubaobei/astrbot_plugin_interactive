# 邮箱系统

## 功能概述
邮箱系统提供用户间异步通信功能，支持发送消息、物品和通知，增强用户体验。

## 命令
- `/mailbox` - 查看收件箱
- `/mail_send [user] [message]` - 发送邮件
- `/mail_read [mail_id]` - 阅读邮件
- `/mail_delete [mail_id]` - 删除邮件

## 功能特性
- 邮件发送接收
- 附件支持
- 邮件分类
- 自动清理

## 邮件类型
- 私人邮件：用户间通信
- 系统通知：系统消息推送
- 活动邀请：活动相关信息
- 交易确认：交易相关通知

## 邮件附件
- 积分附件
- 物品附件
- 代金券附件
- 道具附件

## 安全机制
- 邮件加密传输
- 防垃圾邮件
- 发送频率限制
- 黑名单功能
