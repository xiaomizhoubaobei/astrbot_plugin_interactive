# 股票系统

## 功能概述
虚拟股票系统模拟真实股市，用户可以用积分购买虚拟股票，根据市场波动获得收益或亏损，增加游戏策略深度。

## 命令
- `/stocks` - 查看股票市场
- `/stock_buy [symbol] [quantity]` - 购买股票
- `/stock_sell [symbol] [quantity]` - 出售股票
- `/portfolio` - 查看投资组合

## 功能特性
- 模拟真实股票波动
- 多种虚拟股票
- 实时价格更新
- 投资组合管理

## 股票类型
- 稳健型：波动小，收益稳定
- 成长型：波动中等，潜力较大
- 概念型：波动大，风险与收益并存
- 蓝筹股：大型稳定企业股票

## 市场机制
- 供需关系影响股价
- 新闻事件影响波动
- 宏观经济模拟
- 周期性行业变化

## 投资策略
- 长期持有策略
- 波段操作策略
- 分散投资策略
- 技术分析工具
