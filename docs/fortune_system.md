# 运势系统

## 功能概述
运势系统为用户提供每日运势预测，运势好坏会影响游戏内的各项收益，增加游戏的随机性和趣味性。

## 命令
- `/fortune` - 查看今日运势
- `/fortune_history` - 查看历史运势

## 功能特性
- 每日运势随机生成
- 运势影响多项游戏参数
- 运势详情解析
- 运势提升方法

## 运势等级
- 大吉：所有收益+50%
- 吉利：所有收益+25%
- 一般：正常收益
- 小凶：所有收益-25%
- 大凶：所有收益-50%

## 运势影响范围
- 抽奖收益
- 游戏胜利奖励
- 签到奖励
- 任务奖励
- 牛牛互动效果

## 运势提升
- 使用运势道具提升运势
- 特定时间段运势稳定
- VIP用户运势保底
- 运势兑换积分功能
