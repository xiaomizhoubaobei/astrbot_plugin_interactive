# 公会系统

## 功能概述
公会系统允许用户创建或加入团队，与其他玩家合作完成任务，分享资源，提高游戏乐趣和归属感。

## 命令
- `/guild` - 查看公会信息
- `/guild_create [name]` - 创建公会
- `/guild_apply [name]` - 申请加入公会
- `/guild_leave` - 离开公会
- `/guild_manage` - 公会管理

## 功能特性
- 公会等级系统（通过活动提升）
- 公会技能树（共同研发技能）
- 公会仓库（共享资源）
- 公会任务（团队协作任务）

## 公会管理
- 会长权限：招收成员、踢人、解散公会
- 副会长权限：管理成员、分配任务
- 成员权限：参与活动、使用公会福利

## 公会福利
- 成员积分加成
- 专属公会商店
- 公会活动专属奖励
- 公会技能加成

## 公会建设
- 建设值累积
- 公会建筑升级
- 增加成员上限
- 解锁高级功能
