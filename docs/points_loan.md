# 积分借贷

## 功能概述
积分借贷系统允许用户在积分不足时向系统或其他用户借贷积分，需按时偿还本金和利息，增加游戏策略深度。

## 命令
- `/loan_request [amount]` - 申请贷款
- `/loan_pay [loan_id]` - 偿还贷款
- `/my_loans` - 查看我的贷款

## 功能特性
- 信用评分系统
- 分级利率
- 担保借贷
- 连带责任机制

## 借贷规则
- 最低借款金额：100积分
- 最高借款金额：基于信用评分
- 还款期限：7天/14天/30天
- 利率：5%-20%（基于信用和期限）

## 信用评分
- 基于还款历史
- 借贷频率评估
- 违约记录影响
- VIP身份加分

## 风险控制
- 逾期罚息
- 信用冻结
- 担保人连带责任
- 自动扣款机制
