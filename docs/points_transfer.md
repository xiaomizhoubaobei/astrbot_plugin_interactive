# 积分转账

## 功能概述
积分转账系统允许用户之间直接转移积分，方便玩家之间的资金往来和交易。

## 命令
- `/transfer [user] [amount]` - 转账给指定用户
- `/transfer_history` - 查看转账记录

## 功能特性
- 即时到账
- 支持转账备注
- 转账手续费
- 转账限额控制

## 限制与安全
- 单次转账限额
- 每日转账总额限制
- 验证码确认大额转账
- 转账黑名单功能

## 手续费机制
- 标准用户：转账金额的2%
- VIP用户：转账金额的1%
- 首次转账：免手续费
- 信任用户：降低手续费率

## 转账记录
- 保存所有转账记录
- 支持查询特定时间段
- 可导出转账记录
- 账户余额变动通知
