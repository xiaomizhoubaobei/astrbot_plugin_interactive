# 好友系统

## 功能概述
好友系统允许用户添加其他玩家为好友，进行更深入的社交互动，共享游戏体验。

## 命令
- `/friends` - 查看好友列表
- `/friend_add [user]` - 添加好友
- `/friend_remove [user]` - 删除好友
- `/friend_status` - 设置在线状态

## 功能特性
- 好友状态显示
- 好友互动记录
- 好友分组管理
- 好友推荐系统

## 好友特权
- 好友间赠送无手续费
- 好友互动获得额外经验
- 好友专属任务
- 好友在线提醒

## 互动功能
- 好友间私信
- 一起玩游戏
- 相互协助完成任务
- 好友排行榜

## 隐私设置
- 在线状态控制
- 好友申请审批
- 好友消息筛选
- 好友权限分级
