# 小游戏合集

## 功能概述
小游戏合集包含多种简单有趣的休闲小游戏，如石头剪刀布、掷骰子、猜拳等，增加互动趣味性。

## 命令
- `/rps [choice] [bet]` - 石头剪刀布
- `/dice [bet]` - 掷骰子
- `/coin_flip [choice] [bet]` - 猜硬币
- `/games` - 查看所有小游戏

## 功能特性
- 多种经典游戏集成
- 支持积分投注
- 实时对战模式
- 排行榜统计

## 游戏说明
- 石头剪刀布：选择石头、剪刀或布，获胜获得积分
- 掷骰子：预测骰子点数，越接近真实点数赢得越多
- 猜硬币：猜测正反面，增加简单刺激的赌博元素

## 风险控制
- 单次投注上限
- 每日游戏次数限制
- 防沉迷机制
- 随机性算法保证公平

## 奖励机制
- 基础游戏：低风险低回报
- 高级游戏：高风险高回报
- 连胜奖励：连续获胜额外加成
