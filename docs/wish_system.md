# 许愿池

## 功能概述
许愿池系统允许用户投入积分许下愿望，有机会获得丰厚回报，增加游戏趣味性和期待感。

## 命令
- `/wish` - 许愿
- `/wish_pool` - 查看许愿池
- `/wish_history` - 查看许愿记录
- `/wish_prediction` - 预测中奖概率

## 功能特性
- 许愿池累计奖金
- 随机奖励机制
- 愿望公示
- 中奖公告

## 许愿类型
- 积分许愿：投入积分祈求回本
- 物品许愿：祈求特定物品
- 运势许愿：祈求好运加持
- 成就许愿：祈求达成目标

## 概率机制
- 投入越多中奖概率越高
- 连续许愿有额外奖励
- 高额许愿有保底机制
- 特殊时段概率翻倍

## 池水机制
- 池中积分用于奖励发放
- 一定周期清空池水
- 池水透明度公示
- 池水增长加速
