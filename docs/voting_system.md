# 投票系统

## 功能概述
投票系统允许用户参与社区投票，对特定议题发表意见，影响游戏发展方向，增强社区参与感。

## 命令
- `/vote` - 查看当前投票
- `/vote_cast [poll_id] [option]` - 投票
- `/vote_create` - 创建投票（权限限制）
- `/vote_results [poll_id]` - 查看投票结果

## 功能特性
- 多种投票类型
- 匿名投票选项
- 实时投票统计
- 投票历史记录

## 投票类型
- 功能决策：决定新功能开发方向
- 活动策划：选择下次活动类型
- 规则修改：调整游戏规则
- 争议仲裁：处理社区争议

## 投票机制
- 权重投票（基于活跃度/贡献度）
- 限时投票
- 多轮投票
- 撤销投票（在截止前）

## 社区治理
- 重要决策透明化
- 玩家参与游戏治理
- 民主决策机制
- 投票结果公示
