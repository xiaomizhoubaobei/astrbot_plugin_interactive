# 节日活动系统

## 功能概述
节日活动系统根据现实节日或自定义节日举办特殊活动，提供限时奖励和特色玩法，增加游戏的节庆氛围。

## 命令
- `/event` - 查看当前活动
- `/event_join [event_id]` - 参与活动
- `/event_calendar` - 查看活动日历

## 功能特性
- 自动识别节日
- 限时活动
- 特殊奖励
- 主题装饰

## 活动类型
- 传统节日：春节、中秋、国庆等
- 网络节日：光棍节、双十一等
- 游戏生日：周年庆、上线纪念日
- 特殊主题：夏季活动、冬季活动

## 活动内容
- 限时任务
- 特殊关卡
- 独家奖励
- 主题商店

## 奖励机制
- 活跃度奖励
- 任务完成奖励
- 排行榜奖励
- 随机大奖
