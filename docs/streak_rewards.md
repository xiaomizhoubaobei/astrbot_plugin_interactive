# 签到连击奖励

## 功能概述
签到连击奖励系统在原有签到基础上增加连击概念，连续签到天数越多，奖励越丰厚，鼓励用户持续参与。

## 命令
- `/streak` - 查看当前连击数
- `/streak_history` - 查看历史连击记录

## 奖励机制
- 3天连击：额外+10积分
- 7天连击：额外+30积分 + 特殊道具
- 15天连击：额外+50积分 + 稀有道具
- 30天连击：额外+100积分 + 专属称号
- 60天连击：额外+200积分 + 限定皮肤

## 连击保护
- 可使用连击保护道具避免中断
- VIP用户享受每日一次自动保护
- 补签功能（有限次数）

## 断连惩罚
- 连击中断后降至前一天奖励
- 连续中断超过3天，需要重新开始
- 长期连击中断给予安慰奖

## 特殊奖励
- 月度连击冠军：额外奖励
- 季度连击冠军：专属奖励
- 连击里程碑：特殊成就解锁
