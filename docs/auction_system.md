# 拍卖系统

## 功能概述
拍卖系统允许用户挂售稀有物品或虚拟资产，通过竞价方式出售，促进物品流通和经济系统平衡。

## 命令
- `/auction` - 浏览当前拍卖
- `/auction_sell [item] [start_price]` - 挂售物品
- `/bid [auction_id] [price]` - 出价
- `/my_auctions` - 查看我的拍卖

## 功能特性
- 定时拍卖（24小时倒计时）
- 实时出价通知
- 保证金制度
- 拍卖历史记录

## 拍卖规则
- 起拍价设定（不低于物品价值的30%）
- 加价幅度限制
- 自动延时（最后5分钟有人出价延长5分钟）
- 成交后自动转账

## 拍卖费用
- 挂售手续费：成交价的5%
- 未成交退还手续费：成交价的2%
- 加急上架服务（付费）

## 物品限制
- 只有稀有物品可拍卖
- 不可重复上架相同物品
- 稀缺道具限量拍卖
