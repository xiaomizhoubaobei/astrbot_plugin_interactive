# 谚语填空 - 开发文档

## 功能概述

谚语填空是一个文化教育类游戏，用户需要补全谚语的缺失部分。具有教育意义，实现简单，可增加积分奖励。

## 需求分析

### 功能需求

- 谚语数据库：≥200条谚语
- 填空模式：隐藏部分词汇
- 提示机制：提示缺失字
- 积分奖励：难度决定奖励

## 技术设计

### 数据结构

```python
{
    "id": "str",
    "text": "str",            # 完整谚语
    "missing": "str",         # 缺失部分
    "hint": "str",            # 提示
    "difficulty": "int"       # 难度
}
```

### 模块设计

```
proverb_fill/
├── __init__.py
├── data/
│   └── proverb_database.py   # 谚语库
└── command/
    └── proverb_command.py    # 命令处理
```
