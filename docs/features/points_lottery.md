# 积分商城抽奖 - 开发文档

## 功能概述

积分商城抽奖是现有抽奖系统的扩展，提供不同价位的抽奖池，限定物品和保底机制，促进积分消费。

## 需求分析

### 功能需求
- 多价位池：10/50/100积分
- 限定物品：特殊稀有物品
- 保底机制：必出SR/SSR
- 活动抽奖：限时特殊池

## 技术设计

### 数据结构

```python
{
    "pool_id": "str",
    "cost": "int",            # 抽奖消耗
    "pool_name": "str",
    "items": "list",
    "pity_system": "dict",    # 保底配置
    "is_limited": "bool"      # 是否限定
}
```
