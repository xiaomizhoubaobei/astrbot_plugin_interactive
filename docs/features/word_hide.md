# 隐藏词游戏 - 开发文档

## 功能概述

隐藏词游戏通过emoji提示猜测隐藏词汇，类似猜词游戏。可用emoji提示，支持分类系统。

## 需求分析

### 功能需求
- 隐藏词汇：用**代替部分字
- Emoji提示：相关emoji提示
- 分类系统：动物、食物、日常等
- 提示机制：使用提示令牌
