# 天气预报系统

## 功能概述
天气预报系统模拟真实天气，根据不同的天气状况影响游戏内部分功能的奖励，增加游戏的真实感和变化性。

## 命令
- `/weather` - 查看当前天气
- `/weather_forecast` - 查看天气预报

## 功能特性
- 模拟真实天气变化
- 天气影响游戏奖励
- 一周天气预报
- 特殊天气事件

## 天气类型与效果
- 晴天：抽奖奖励+5%
- 雨天：签到奖励+10%
- 雪天：牛牛互动效果翻倍
- 雾天：猜数字游戏奖励+15%
- 雷雨天：所有奖励+20%
- 彩虹天：特殊奖励（罕见天气）

## 天气预测
- 基于历史数据预测
- 天气趋势提醒
- 特殊天气预警
- 天气变化奖励

## 用户互动
- 分享天气截图
- 天气话题讨论
- 天气相关任务
- 服装搭配系统
