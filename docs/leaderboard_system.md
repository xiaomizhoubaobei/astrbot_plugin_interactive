# 排行榜系统

## 功能概述
排行榜系统允许用户查看服务器中最活跃、最有成就的玩家。显示多种排名指标，包括总积分、签到次数、抽奖次数等。

## 命令
- `/leaderboard` - 查看综合排行榜
- `/leaderboard points` - 查看积分排行榜
- `/leaderboard signs` - 查看签到排行榜
- `/leaderboard lotteries` - 查看抽奖排行榜

## 功能特性
- 实时更新排名
- 前三名显示特殊标识
- 可查看自己在排行榜上的位置
- 支持分页显示

## 数据统计
- 自动记录用户的积分变化
- 追踪用户的各种活动次数
- 每小时更新一次排名数据

## 权限设置
- 所有用户均可查看排行榜
- 隐藏用户可选择不在排行榜上显示
