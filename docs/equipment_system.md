# 装备系统

## 功能概述
装备系统允许用户为角色或宠物装备不同属性的装备，提升在游戏中的能力表现。

## 命令
- `/equipment` - 查看装备
- `/equip [item_id]` - 装备物品
- `/unequip [slot]` - 卸下装备
- `/equipment_upgrade [item_id]` - 升级装备

## 功能特性
- 多种装备槽位
- 属性加成系统
- 装备合成
- 强化升级

## 装备类型
- 武器：增加攻击力
- 防具：增加防御力
- 饰品：提供特殊效果
- 宠物装备：提升宠物能力

## 装备品质
- 普通：白色（+0%属性）
- 精良：绿色（+10%属性）
- 稀有：蓝色（+25%属性）
- 史诗：紫色（+50%属性）
- 传说：橙色（+100%属性）

## 强化系统
- 强化成功率
- 强化失败保护
- 强化等级上限
- 强化继承机制
